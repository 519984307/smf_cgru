CGRU Utilities.

Keeper

System tray menu to launch all CGRU applications.

Afanasy Starter

Stand alone dialog to submit jobs to Afanasy.

Movie Maker

Create movie file from image sequence. Consists of a Python command and a GUI (PyQt) for it.

Regular Expressions

Regular expressions examples. Links on detailed documentation.

RegExp Checker

Simple dialog to check regular expressions.

Maintenance

Utilities for CGRU project maintenance. Does not needed in releases. Available in the project repository only.

Release

Utility to create archives for all supported platforms, 'deb' and 'rpm' packages.

Site

Parse all CGRU 'html' files to replace header and footer for local and remote version.
