CGRU

Make Movie.

This is an utility to create movie file from image sequence. It consists of command and GUI for it. GUI is written on PyQt, so python and PyQt must be installed. Command is written on python. It launches other utilities: ImageMagick to draw annotations and ffmpeg to encode. On Linux and Windows ImageMagick must be installed. To encode 'H.264' codec it uses ffmpeg with x264 library. On Windows ImageMagick contains ffmpeg binary compiled with x264 library. Linux releases contains ffmpeg binary compiled with x264 library.
