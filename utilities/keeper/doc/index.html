CGRU Keeper.

A system tray icon menu to launch and manage CGRU applications, launch and configure other software.

It is also designed to unify menus on all supported platforms: Linux, MS Windows and Mac OS X.

Software Menu: Afanasy Menu: MovieMaker Menu:

[ browse ] - Open a folder that contains launch scripts. Useful to create links on a desktop.

Launch:

  • MS Windows: cgru\start.cmd
  • Mac OS X: cgru/start.sh
  • Linux: cgru/start.sh
