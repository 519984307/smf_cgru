CGRU Installation

Get CGRU

Download a latest release and unpack.

You can use SVN to check out latest version:

$ svn co https://cgru.svn.sourceforge.net/svnroot/cgru cgru

And build it yourself.

Debian and RPM Packages.

For Linux with Debian (Debian, Ubuntu, ...) and RMP (RedHat, CentOS, Fedora, OpenSUSE, ...) Afanasy has software packages.
See this Linux Packages page.

Setup Environment

You can setup you $PATH and $PYTHONPATH variables, to use Afanasy commands and to import Python module.

To setup CGRU commands and Python module, you need to go in CGRU folder and source a script:

$ cd cgru
$ source setup.sh

To setup Afanasy commands and Python module, you need to go in Afanasy folder and source a script:

$ cd cgru/afanasy
$ source setup.sh

You can put next lines at the end of your ~/.bashrc file:
# CRGU:
pwd=$PWD; cd cgru; source ./setup.sh; cd $pwd
# Afanasy:
pwd=$PWD; cd cgru/afanasy; source ./setup.sh; cd $pwd
 And shell will automatically source it at every launch.

MS Windows: cmd scripts are also provided (its do the same).

Run CGRU

Afanasy examples are located in cgru/afanasy/examples directory.
General tools for specific soft are located in cgru/plugins/[software_name] folder.
Afanasy Tools for specific soft are located in cgru/afanasy/plugins/[software_name] folder.
Examples has software launching scripts to make them to run with Afanasy setup.
Documentation for specific soft in located in cgru/plugins/[software_name]/doc folder.
Run examples by executing script from it example directory.
For example to launch Houdini example:
$ cd .../cgru/afanasy/examples/houdini
$ ./houdini

MS Windows: You can double click application launch script in cgru\bin directory, or create link to it.
To launch example run its script. Example for NUKE: cgru\afanasy\examples\nuke\nuke.cmd
