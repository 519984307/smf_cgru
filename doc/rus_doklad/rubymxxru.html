Ruby - Mxx_ru.

Ruby - объектно орииентированный язык програмирования, интерпретатор. Кросс-платформенный.

Mxx_ru - библиотека классов Ruby (Ruby Gem). Служит для сборки С++ программ. Он анализирует директории, генерит и запускает командные строки для компиляции и последующей линковки. На нём можно написать небольшой скрипт, который будет собирать ваш проект под любую OS и многие известные компиляторы. Так же есть поддержка генераторов кода, например "moc" файлы Qt.

Удобен для кросс-платформенных проектов. Разрабатывать можете в любой OS, используя любимое IDE. А небольшой скрипт на Ruby всё собирает. Вы можете сказать, что IDE всё сам может собирать. Хорошо отделить IDE от сборки. Если на проекте несколько программистов, желающие сидеть каждый на своём любим IDE - тоже пожалуйста, пусть каждый пишет где и как ему удобно.
