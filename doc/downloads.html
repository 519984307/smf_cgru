  • CentOS 5 (RHEL 5)
    Warning! Compiled with Python 2.7.2. CentOS 5 native Python version is 2.4 which is not compatible with Afanasy.
    Installation of Python2.7.2 can damage CentOS 5. Better just to download and compile Python 2.7.2 and to setup environment for clients to use it.
    Qt version is 4.7.4 static.
  • GCC-4.1 Python-2.7.2 Qt-4.7.4-static
