Softimage Python Setup.

Softimage 2010 has no internal Python, so to use it you should setup it. Softimage 2011 has internal Python, but you can setup it to use your own, your version with your additional packages.

Install Python.
Install Python For Windows.
Normally it should appear in Softimage.
If not you can run
Python25\Lib\site-packages\win32comext\axscript\client\pyscript.py
script to register it.

You can simple install all it only on one machine and just copy to anothers. But you must run register script in this case.
