Сайт

1. Скачать.

2. Распаковать/положить куда-нибудь.

3. Скопировать запускной скрипт, находящийся в cgru/maya/maya.bat ( или maya.sh для linux ), например на рабочий стол ( скрипт будет настраивать маю и cgru перед запуском майи).

4. Настроить скрипт:

поправить на то куда вы переписали cgru и где у вас стоит майа, какая версия и сколько бит, например:

версия майи
set MAYA_VERSION=2009
set MAYA_ARCH=-x64
 rem For 32bit Maya uncomment next line
удалить rem перед следующей строчкой, если майа 32-ух битная
rem set MAYA_ARCH=
где находится cgru-шные тулзы для маи
set MAYA_CGRU_LOCATION=c:\cg\tools\cgru\plugins\maya
где находится maya
set MAYA_LOCATION=c:\autodesk\Maya2009
