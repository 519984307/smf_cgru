getenv MAYA_SCRIPT_PATH, MAYA_PLUG_IN_PATH, MAYA_MODULE_PATH, XBMLANGPATH.

It is a simple command that splits the result of maya command "getenv" and outputs ot in a scriptEditor for more comfortable reading. Very useful for debugging when you have lots of tools paths.

Exapmle:

getenv MAYA_SCRIPT_PATH result:

// Result: //Server2/projects/Turecky_Gambit;/scripts;//server2/tools/tools_maya/overrideScripts65;//server2/root/Tools/tools_maya/main/scripts;C:/AW/tools_maya;//Server2/usr/diaxe/.profile/Maya/6.0/scripts;C:/_USERS_DATA_/_TIMA_DATA_/maya/6.5/scripts;C:/_USERS_DATA_/_TIMA_DATA_/maya/scripts;C:/_USERS_DATA_/_TIMA_DATA_/maya/6.5/presets;C:/_USERS_DATA_/_TIMA_DATA_/maya/6.5/prefs/shelves;C:/_USERS_DATA_/_TIMA_DATA_/maya/6.5/prefs/markingMenus;C:/AW/Maya6.5/scripts/startup;C:/AW/Maya6.5/scripts/others;C:/AW/Maya6.5/scripts/AETemplates;C:/AW/Maya6.5/scripts/paintEffects;C:/AW/Maya6.5/scripts/fluidEffects;C:/AW/Maya6.5/scripts/hair;C:/AW/Maya6.5/scripts/cloth;C:/AW/Maya6.5/scripts/live;C:/AW/Maya6.5/scripts/fur;//server2/3d/Tools/tools_maya/bonusTools6.5/scripts;//server2/3d/Tools/tools_maya/main/scripts;//server2/3d/Tools/tools_maya/stuff/scripts;//server2/3d/Tools/tools_maya/cps6/scripts //
