MAX Afanasy Setup

Afanasy MAX max submission script is located in afanasy/plugins/max directory. It cab be called through CGRU menu. CGRU and Afanasy environment must be set.

MAX Setup Example:

You can use 3dsmax and 3dsmaxcmd commands, which call the same setup script.

3dsmax.cmd


   call setup_max.cmd

   IF [%1]==[] (
   set TOSTART=start
   ) ELSE (
   set TOSTART=
   )

   %TOSTART% 3dsmax.exe %*

3dsmaxcmd.cmd


   call setup_max.cmd   

   3dsmaxcmd.exe %*

setup_max.cmd


   call setup_cgru.cmd
   call setup_afanasy.cmd

   set MAX_LOCATION=C:\Program Files\Autodesk\3ds Max 2010   

   PATH=%CGRU_LOCATION%\plugins\max;%PATH%

   if exist override.cmd call override.cmd

   set PATH=%MAX_LOCATION%;%PATH%

Where setup_cgru.cmd and setup_afanasy.cmd are script which setup CGRU and Afanasy in your company, or project, or asset.

Full working example is located in cgru/examples/max.
