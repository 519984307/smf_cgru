Afanasy in Blender

    • Afanasy menu item in main menu
    Create job dialog
    • Start Frame - First frame to render.
    • End Frame - Last frame to render.
    • Per Host - Number of frames in each task.
    • Max Hosts - Maximum number of host job can take ('-1' means no limit).
    • Priority - Job priority (execution order).
    • Depend Mask - Other job(s) name pattern to wait (empty value means not wait any job).
    • Hosts Mask - Hosts names pattern job tasks can run on (if empty job can run on any host).
    • Hosts Exclude - Hosts names pattern job tasks can not run on.
    • Properties - Hosts properties pattern job tasks can not run on.
    • Start Paused - Start job paused (set off-line state).

    Script detects Blender rendering engine (internal or Yafray) and set special service and parser to get correct task progress. Parser (and service) will be 'blender_internal' or 'blender_yafray' (parsers and services are python scripts too, located in afanasy). They parse and blender and renderer output, so if task consists of several frames, output will be correct too.

    Example

    CGRU Blender Afanasy example located in cgru/examples/blender directory.
    Launch example:

    $ cd cgru/examples/blender
    $ ./blender scene.blend
