XSI Afanasy Setup

Afanasy XSI python plugin is located in cgru/plugins/xsi directory. You can add it automatically by adding this directory to XSI_PLUGINS variable. This plugin imports Afanasy version dependent libraries, located in afanasy/plugins/xsi. So you should add this folder in PYTHONPATH variable.

XSI Setup Example:

You can use xsi and xsibatch commands, which call the same setup script.

xsi.cmd


   call setup_xsi.cmd

   "%XSI_LOCATION%\Application\bin\XSI.bat" %*

xsibatch.cmd


   call setup_xsi.cmd

   "%XSI_LOCATION%\Application\bin\XSIBatch.bat" %*

setup_xsi.cmd


   call setup_cgru.cmd
   call setup_afanasy.cmd

   set XSI_CGRU_PATH=%CGRU_LOCATION%\plugins\xsi
   set XSI_AF_PATH=%AF_ROOT%\plugins\xsi

   set XSI_LOCATION=C:\Program Files\Autodesk\Softimage 2011   

   if exist override.cmd call override.cmd

   if defined XSI_PLUGINS (
      set XSI_PLUGINS=%XSI_CGRU_PATH%;%XSI_PLUGINS%
   ) else (
      set XSI_PLUGINS=%XSI_CGRU_PATH%
   )

   if defined PYTHONPATH (
      set PYTHONPATH=%XSI_AF_PATH%;%PYTHONPATH%
   ) else (
      set PYTHONPATH=%XSI_AF_PATH%
   )

Where setup_cgru.cmd and setup_afanasy.cmd are script which setup CGRU and Afanasy in your company, or project, or asset.

The full working example is located in afanasy/examples/xsi.
