Afanasy MS Windows Client

There is no server for windows. Afanasy server uses some Linux common tools, for example Apache, SQL (Postgre), PHP. There is no reason to translate sources or launch with emulators current Linux version. Any (even very small) company can have at least one Linux server for various purposes (forum, web, ftp, mail, file server, router). You can use it for Afanasy too.

Latest release contains windows client applications: render, watch, talk and python module to send jobs.

You can build the latest version yourself using MSVC. Dowload and install Qt. Use SVN (Subversion) to get code and CMake to build MSVC project files.

  1. Get CGRU

    Download the latest release and unpack.

  2. Install python2.5

    Afanasy is built with Python 2.5 version. To use other python versions you can download bin_pyaf archive. It contains pyaf.pyd Afanasy python plugin for various python versions, bits and OS'es.

  3. Launch

    Afanasy applications launch scripts are located in cgru/examples/scripts directory.
    afrender.cmd - render slave client.
    afwatch.cmd - GUI.
    aftalk.cmd - chat client.
    All example scripts use cgru/examples/setup_cgru.cmd and cgru/examples/setup_afanasy.cmd scripts. They show how to setup CGRU and Afanasy.
    To setup CGRU call setup.cmd in cgru folder.
    To setup Afanasy call setup.cmd in afanasy folder.

  4. Web Server

    If it installed on afanasy server, it available at

    http://[afanasy server computer name, or IP address]

    Or ask your admin configured it.
