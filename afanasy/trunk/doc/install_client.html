Afanasy Client

  1. You need UNIX

    Any Linux or Mac or other UNIX (any POSIX, so may be MS Windows with emulators, if you are brave).

  2. Required packages:

    libqt4-gui
    python2.5

  3. Get CGRU

    The best way to have the same directory:
    /cg/tools
    Setup you network to make sever to export this directory.
    Mount it at every client computer.

  4. Setup Environment

    See previous documentation page.

  5. Web Server

    If it installed on afanasy server, it available at

    http://[afanasy server computer name]
