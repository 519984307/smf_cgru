Doxygen

Doxygen is a source code documentation generator tool.

To make following links working you must build HTML pages by doxygen.

  • Afanasy - Main server and general library, SQL, networking, CLI.
  • Render - Render client host and general library, Qt networking.
  • Watch - Watch GUI and general library, Qt networking.

Build commands:

$ cd cgru/afanasy/doc/doxygen
$ doxygen afanasy
$ doxygen render
$ doxygen watch
