Afanasy Web Visor

Web Interface to Afanasy SQL database

You can watch all users jobs, users activity and various statistics tables and diagrams.

All you need is any web browser and connection with afanasy server. No database connection or browser plugins need, all calculations on server side, you browser will receive plain html text and png images.

Web Browser Address

Type address in browser:
http://[Afanasy server name]/afanasy/php
or simple this (depends on you web server settings, ask you admin):
http://[Afanasy server name]
may be even this, if name is not available:
http://[Afanasy server IP address]
