In this scene (A_SCENE) all GUI functionality is enabled. But most will not works especially for guests. This is mostly because this server does not have Afanasy, Python and you machine does not have this filesystem.

You can see cmd buttons with yellow back. This bottons are for system commands running. Commands will be executed locally by your browser if cgru-addon installed.

  • Open - open folder in file explorer.
  • Nuke - open sequence in Nuke.
  • NkCp - copy sequence as Nuke Reader.
  • PdP - open sequence in PdPlayer.
  • Pd++ - add sequence in opened PdPlayer.
Command buttons can be customized. You can run any commands with a path as argument.
